# CSKZ-AutoBhopJump


### BhopJump

#### 半联动模式:判断当前鼠标位置来判断发送左或右键代码实现空中转向加速 同时不停的发送滚轮跳来实现BJ

#### 在FPS稳定的情况下 可以手动空中最多加速4次 空速可以稳定在320左右

#### 超过4次空中加速脚本就跟不上了
